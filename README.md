# MHT-CET Test Portal

A responsive and user-friendly MHT-CET Test Portal built using HTML, CSS, and JavaScript. The application allows users to select subjects and chapters, attempt timed quizzes, navigate between questions, and view instant results with scoring.

#Features:

•Subject selection (Physics, Chemistry,  Mathematics)
•Chapter-wise MCQ tests
•Timer for each test
•Previous/Next navigation
•Instant result calculation
•Clean and responsive UI

#Technologies Used:

•HTML
•CSS
•JavaScript

#How to Run Project:

1.Download or clone this repository  
2.Extract the files (if downloaded as ZIP)  
3.Make sure all files are in the same folder  
4.Open `index.html` in any web browser (Chrome recommended)


#Project Structure:

•index.html → Main structure
•style.css → Styling
•script.js → Functionality and question logic
